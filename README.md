			<li>Try to make it out alive!</li>
		</ul>
		<h2>Features</h2>
		<ul>
			<li>Randomized monster encounters for added replayability</li>

			<li>A creepy atmosphere that will keep you on the edge of your seat</li>
		</ul>

		</ol>
